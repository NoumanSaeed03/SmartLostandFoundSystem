# 🔍 Smart Lost & Found System

A web-based application designed to help users report lost items, submit found items, and reconnect people with their belongings through an intelligent matching system.

---

## 🚀 Features

- 📌 Report Lost Items
- 📌 Report Found Items
- 🤖 Smart Matching (based on location, category, and date)
- 🔐 User Authentication & Secure Access
- 📊 Organized Item Management
- 💬 Easy communication between users

---

## 🛠️ Technologies Used

- ASP.NET Core MVC
- C#
- SQL Server
- Bootstrap (Frontend UI)
- Entity Framework

---

## 💡 How It Works

1. Users report a lost or found item
2. System stores item details (category, location, date, etc.)
3. Matching logic finds similar items
4. Users connect and recover belongings

---

## 📁 Project Structure

- Controllers → Handle user requests
- Models → Data structure & database logic
- Views → User Interface (Razor Pages)
- wwwroot → Static files (CSS, Images, JS)

---

## 🔐 Privacy & Security

- Only authenticated users can access the system
- User data is protected and not shared publicly
- Secure handling of item and user information
